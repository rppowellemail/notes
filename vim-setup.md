# expand tabs

https://stackoverflow.com/questions/1562336/tab-vs-space-preferences-in-vim

```
shiftwidth=4
softtabstop=4
tabstop=4
```

```
:e $HOME/.vimrc
set tabstop=4
set shiftwidth=4  "tabs are 4 spaces wide (default = 8)
set expandtab     "Convert tabs to spaces
```
